Uso de Decimal Pipe

Aquí se pone un número sin formato dado: {{number_decimalPipe}}

Aquí se pone el mismo número aproximado a la centésima mediante Decimal Pipe: {{number_decimalPipe | number:'3.0-2'}}

Uso de Slice Pipe

Aquí se imprime un array sin hacer nada: {{animals}}

Aquí se imprime un array usando Slice Pipe para solo imprimir tres: {{animals | slice:0:3}}

Uso de Currency Pipe

Aquí se imprime un precio sin formato: {{price}}

Aquí se imprime un precio con símbolo de libras en formato español con decimales truncados usando Currency Pipe: {{price | currency: 'symbol':'1.0-2':'GBP'}}

Uso de Date Pipe

Aquí se imprime una fecha con el formato por defecto: {{date}}

Aquí se imprime una fecha con formato 'longDate': {{date | date:'longDate'}}
